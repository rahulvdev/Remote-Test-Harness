OPERATION SPECIFICATIONS:
------------------------
1)When run.bat executes, or when the solution is executed on VISUAL STUDIO, the test executive runs and the results are displayed on
  the user interface.
2)When the "Browse DLL Path" button is clicked, the FilesToRepository 'folder' must be chosen, since all the library files are stored in that folder.
3)When the "Browse Request File" button is clicked, select xml file from the 'TestRequest' folder.
4)For fetching logs, 2 parameters are needed, i.e Author and Test Request name.These 2 parameters are initially used to specify log file name.
  *****Author name = Rahul Vijaydev
  *****TestId = FirstTestRequest
For other cominations we get a "Log not found message on the User Interface.
